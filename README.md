# Breakout_Reborn

BMAD 기반으로 진행하는 `Breakout_Reborn` 게임 프로젝트입니다.

## 구조

- `_bmad-output/`: 기획, 구현, 테스트 산출물
- `app/`: 게임 애플리케이션 코드
- `CLAUDE.md`: Claude 계열 에이전트 작업 가이드
- `GEMINI.md`: Gemini 계열 에이전트 작업 가이드

## 권장 시작 순서

1. `gds-create-game-brief`
2. `gds-create-gdd`
3. `gds-game-architecture`
4. `gds-create-epics-and-stories`
5. `gds-check-implementation-readiness`
6. `gds-sprint-planning`
