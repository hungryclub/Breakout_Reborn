# Story 2.3: 파워업 드롭과 획득 루프 구현

Status: review

## Story

As a 플레이어,  
I want 아이템 브릭에서 파워업이 떨어지고 패들로 직접 획득하기를,  
so that 같은 런 안에서도 리스크와 보상이 섞인 변주를 체감할 수 있다.

## Acceptance Criteria

1. 아이템 브릭 파괴 시 낙하형 파워업 오브젝트가 생성되어야 한다.
2. 파워업 오브젝트는 공의 메인 판독성을 방해하지 않는 속도로 아래로 떨어져야 한다.
3. 패들이 파워업을 획득하면 획득 이벤트가 발생하고 UI에 짧은 반응이 보여야 한다.
4. 획득하지 못한 파워업은 화면 아래로 벗어나면 자연스럽게 제거되어야 한다.
5. 드롭과 획득 규칙은 시스템과 순수 함수로 분리되어 테스트 가능해야 한다.

## Dev Notes

- 이 단계에서는 “효과가 실제로 적용된다”보다 “드롭과 수집의 물리 루프가 안정적으로 돈다”가 더 중요하다.
- 다음 스토리에서 4종 파워업 효과를 붙일 수 있도록 타입 값과 이벤트 계약을 먼저 고정한다.
- 드롭 속도는 과하게 빠르지 않되 공의 메인 판독성을 해치지 않는 선으로 유지한다.
