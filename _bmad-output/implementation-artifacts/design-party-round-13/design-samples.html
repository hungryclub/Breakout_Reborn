Breakout Reborn / Design Party Round 13

브러시 질감과 노이즈를 더 밀어 넣은 손그림 풍 콘셉트아트 접근

이번 라운드는 형태를 깔끔하게 닫지 않고, 표면 질감과 안개와 빛 번짐 속에서 덩어리가 드러나도록 설계했습니다. 정리된 UI 포스터보다는 아트북에 실린 콘셉트 샷처럼 보이도록, 화면 가장자리와 경계를 더 흐리고 환경과 존재가 서로 섞여 읽히게 만들었습니다.

Sketch Rules
1. 덩어리는 선보다 공기층과 명암으로 읽힌다
2. 기하학적 대칭보다 붕괴와 치우침을 우선한다
3. 광원은 점이 아니라 퍼지는 현상처럼 다룬다
4. 캐릭터와 풍경은 분리된 요소가 아니라 한 장면 안에서 섞인다
Round 13 / Smoke Beast Painting
SKETCH SIEGE
Heart Gate Siege / Hand-painted Fortress Beast

Let the Warden Appear Like a Beast Buried in the Citadel

이 버전에서는 수문장을 완전한 중심 캐릭터로 세우지 않고, 붕괴하는 성채 입구와 연기층 속에 박혀 있는 금속 짐승처럼 처리했습니다. 얼굴과 코어는 화염과 먼지 속에서 반쯤 드러나고, 전경의 광탄 궤적과 문 파편이 충돌 순간의 사건감을 끌어올립니다.
This should feel like the enemy is emerging from the ruin itself.
Sketch Tone
Ruin / Smoke / Collision
Round 13 / Mythic Shrine Painting
SKETCH ECLIPSE
Last Eclipse / Hand-painted Dawnheart

Let the Dawnheart Wake Like a Weather Event Over Ancient Ruins

이 버전은 중심 상징을 더 줄이고, 절벽과 성소와 하늘과 안개가 함께 반응하는 풍경으로 재구성했습니다. 심장 존재는 풍경 한가운데서 숨을 되찾고, 빛은 직선 장치가 아니라 공기 속에서 퍼지는 현상처럼 다뤄져 조금 더 손그림 풍 장면으로 읽히게 합니다.
This should feel like dawn is returning to the world, not just to the object.
Sketch Tone
Mist / Shrine / Dawn
