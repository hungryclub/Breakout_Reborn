# Breakout Reborn 디자인 파티 라운드 3

## 목적

이번 라운드의 목표는 단순히 고급스럽고 예쁜 브레이크아웃 화면을 만드는 것이 아니라, 화면만 봐도 아래 3가지가 즉시 읽히는 시안을 만드는 것이다.

1. 플레이어가 누구인가
2. 지금 무엇을 해야 하는가
3. 성공하면 세계가 어떻게 변하는가

사용자 피드백의 핵심은 분명했다. 기존 시안들은 블록에 색을 입힌 수준이었고, 게임의 테마와 목표, 그리고 게임성이 화면에 드러나지 않았다. 따라서 이번 라운드에서는 스타일보다 먼저 `미션 가독성`, `세계관 오브젝트화`, `진행의 시각화`를 우선 기준으로 삼았다.

## 공통 진단

모든 에이전트가 공통으로 지적한 문제는 다음과 같다.

- 브릭이 모두 같은 급의 장식으로 보여서 왜 깨야 하는지 의미가 없다.
- 패들과 공이 세계 안의 도구가 아니라 단지 UI 오브젝트처럼 보인다.
- HUD가 점수만 보여줄 뿐 현재 미션의 진척과 위기, 보상을 설명하지 못한다.
- 스크린샷 한 장만 봐서는 어떤 테마의 어떤 게임인지 유저가 이해하기 어렵다.
- 따라서 “예쁘다”는 감상은 남아도 “해보고 싶다”는 욕망으로 이어지지 않는다.

## 에이전트별 논의

### 웹 디자인 에이전트

웹 디자인 에이전트는 이번 실패의 본질을 `역할의 문법 부재`로 정의했다. 좋은 브레이크아웃 시안은 배경이 예쁜 화면이 아니라, 지금 무엇을 왜 어떻게 부수는지 즉시 말하는 화면이어야 한다고 보았다.

주요 제안:

1. `Orbital Salvage`
2. `Citadel Breach`
3. `Temple Awakening`
4. `Neon Heist Run`
5. `Garden Rebirth`

핵심 메시지:

> 좋은 breakout 테마는 예쁜 배경이 아니라, 지금 무엇을 왜 어떻게 부수는지 즉시 말하는 화면이어야 한다.

### 아트 디자인 에이전트

아트 디자인 에이전트는 `세계의 문법`이 화면 안에서 읽혀야 한다고 강조했다. 브릭은 세계의 구조물이어야 하고, 패들은 도구여야 하며, 공은 행동의 의미를 가진 핵심 오브젝트여야 한다고 봤다.

주요 제안:

1. `Bio Ark Revival`
2. `Salvage Launchyard`
3. `Eclipse Sanctum`
4. `Abyss Pressure Dome`
5. `Stormline Courier`

아트 디자인 에이전트는 특히 `Eclipse Sanctum`의 비주얼 임팩트, `Salvage Launchyard`의 상업성, `Abyss Pressure Dome`의 개성을 높게 평가했다.

### 2D 게임 디자인 에이전트

2D 게임 디자인 에이전트는 유저가 첫 발사 순간에 목표를 이해할 수 있어야 한다는 점을 가장 강하게 주장했다. 벽을 부순다는 행위가 곧 세계의 장애물을 해제하는 행위로 읽혀야 하고, HUD는 그 진척을 설명해야 한다고 봤다.

주요 제안:

1. `성채 돌파`
2. `우주 정거장 구조`
3. `성소 봉인 해제`
4. `산호초 복원`
5. `하늘정원 개화`

2D 게임 디자인 에이전트는 `성채 돌파`를 가장 강한 게임 읽힘, `우주 정거장 구조`를 가장 빠른 유저 호감 방향으로 평가했다.

### 3D 게임 디자인 에이전트

3D 게임 디자인 에이전트는 기존 시안들이 실패한 이유를 `어느 세계의 어떤 순간인지 말해주지 못한 것`이라고 정의했다. 카메라, 깊이, 재질, 환경 오브젝트가 모두 합쳐져야 목표가 보인다고 보았다.

주요 제안:

1. `Orbital Beacon Rebuild`
2. `Eclipse Temple Unseal`
3. `Sunken Archive Recovery`
4. `Clockwork Rail Run`
5. `Genesis Garden`

3D 게임 디자인 에이전트는 스토어 페이지 한 컷의 힘은 `Orbital Beacon Rebuild`가 가장 세다고 보았다.

### 게임 유저 에이전트

게임 유저 에이전트는 가장 직설적이었다. 기존 시안은 “예쁜 화면”이었지만 “게임”은 아니었다고 평가했다. 유저는 색이 예쁜 것보다 `쏘고`, `깨고`, `구하고`, `열고`, `되살리는` 동사를 먼저 본다고 정리했다.

주요 제안:

1. `Orbit Rescue`
2. `Vault Break`
3. `Reef Rebirth`
4. `Sky Temple Awakening`
5. `Neon Grid Defense`

게임 유저 에이전트는 가장 먼저 설치해볼 게임으로 `Orbit Rescue`를 꼽았다.

## 종합 결론

각 에이전트가 사용한 명칭은 조금씩 달랐지만, 실제로는 다음 다섯 방향으로 수렴했다.

1. 우주 구조/복구
2. 성채/금고 돌파
3. 신전/성소 봉인 해제
4. 심해 돔/수중 구조
5. 생태 복원/생명 재생

이번 라운드 최종 시안은 위 다섯 방향 중, 사용자 요구와 게임 화면의 가독성을 모두 만족시키는 조합으로 다음 다섯 콘셉트를 채택했다.

## 최종 채택 시안

### 1. Orbital Rescue Beacon

- 테마: 고장 난 궤도 정거장과 비콘 복구
- 목표: 모듈을 복구하고 도킹 비콘을 재점화
- 브릭 역할: 파손 패널, 전력 셔터, 잔해 벽
- 패들 역할: 자기장 수리 레일
- 볼 역할: 구조 드론 펄스
- 성공 후 변화: 경고등이 꺼지고 정거장 조명이 살아남

### 2. Citadel Breach

- 테마: 적 성채 공성전
- 목표: 외벽과 방패층을 파괴해 내부 코어를 노출
- 브릭 역할: 외벽, 문장판, 방어막 노드
- 패들 역할: 공성 발사대
- 볼 역할: 브리치 오브
- 성공 후 변화: 성문이 열리고 중앙 홀로 진입

### 3. Eclipse Sanctum

- 테마: 일식 속 봉인된 성소 해방
- 목표: 룬 봉인을 모두 해제해 태양 코어를 각성
- 브릭 역할: 봉인석, 룬 패널, 제단 방벽
- 패들 역할: 의식 거울 제단
- 볼 역할: 태양 파편 구슬
- 성공 후 변화: 봉인이 열리고 중심광이 폭발적으로 깨어남

### 4. Abyss Pressure Dome

- 테마: 심해 도시 붕괴 방지
- 목표: 압력 돔의 균열과 밸브 장애물을 제거해 도시를 안정화
- 브릭 역할: 금 간 벌크헤드, 밸브 잠금, 산호화 장벽
- 패들 역할: 소나 수리 장벽
- 볼 역할: 수리 드론 코어
- 성공 후 변화: 돔의 압력 경고가 안정 상태로 바뀌고 도시 조명이 회복

### 5. Bio Ark Revival

- 테마: 죽은 행성의 생명 재생
- 목표: 오염층을 벗기고 씨앗 코어를 활성화해 생태계 복원
- 브릭 역할: 화석층, 씨앗 포드, 오염 셸
- 패들 역할: 생장 유도 레일
- 볼 역할: 바이오 시드 코어
- 성공 후 변화: 지표에 녹색 광맥과 새싹이 퍼짐

## 이번 라운드 디자인 원칙

1. 브릭은 장식이 아니라 세계의 장애물이어야 한다.
2. 패들은 스포츠 바가 아니라 세계 안의 도구로 보여야 한다.
3. 공은 단순한 공이 아니라 미션을 수행하는 핵심 매개체여야 한다.
4. HUD는 점수보다 목표, 위기, 진척, 보상을 먼저 보여야 한다.
5. 한 컷만 봐도 유저가 “왜 이걸 해보고 싶은지” 이해할 수 있어야 한다.
