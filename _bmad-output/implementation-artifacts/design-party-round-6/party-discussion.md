# Breakout Reborn 디자인 파티 라운드 6 토론 기록

## 질문

라운드 5의 3안, 4안, 5안을 다음 단계로 밀 때 어떤 점을 더 공격적으로 바꿔야 하는가.

## 에이전트별 의견

### 웹 디자인 에이전트

- 3안은 대칭형 문 아이콘처럼 보여 침투 사건보다 상징이 먼저 읽힌다.
- 4안은 원형 패턴이 퍼즐처럼 보여 침입의 절박함이 약하다.
- 5안은 상징성과 분위기는 좋지만 실제 목표가 늦게 읽힌다.

핵심 제안:
- 3안은 `왼쪽 아래 공격선 -> 오른쪽 거대 관문 -> 뒤 심장실` 흐름으로 바꿀 것
- 4안은 `바깥 경보 확산 -> 잠금 링 -> 중앙 보상 코어` 구조를 만들 것
- 5안은 `상단 일식 -> 중앙 광선 경로 -> 하단 심장` 수직 축을 명확히 할 것

### 아트 디자인 에이전트

- 3안은 금속색 네모 패널로 보여 장갑층의 사건성이 약하다.
- 4안은 경보 테마 원형 퍼즐처럼 보인다.
- 5안은 보라/금색 장식으로 읽혀 점화 의식의 무게가 약하다.

핵심 제안:
- 3안은 그을린 장갑판과 틈 사이 백열 코어를 중심으로 할 것
- 4안은 닫혀가는 셔터와 매달린 기록핵이 핵심 실루엣이 되어야 한다
- 5안은 검은 현무암 제단과 빛이 새는 심장 코어로 재구성할 것

### 2D 게임 디자인 에이전트

- 3안은 장갑층, 방어층, 심장 노출의 3단계가 한눈에 분리되어야 한다.
- 4안은 금고의 잠금 구조와 봉쇄 카운트가 중심에서 읽혀야 한다.
- 5안은 심장을 깨우는 의식 진행과 일식 마감 시간이 더 직접적으로 보여야 한다.

핵심 제안:
- 브릭, 패들, 볼의 역할을 세계 장치로 재해석할 것
- 실패 압박은 숫자가 아니라 장면 안 변화로 보이게 할 것
- 성공 순간은 세계 변화로 보일 것

### 3D 게임 디자인 에이전트

- 3안은 정면 아이콘처럼 보여 공성전 스케일이 약하다.
- 4안은 원형 UI처럼 보여 금고 침투로 안 읽힌다.
- 5안은 상징 문양이 먼저 보여 사건성이 약하다.

핵심 제안:
- 3안은 긴 복도 끝 심장 게이트를 향한 돌파 구도로 만들 것
- 4안은 좌우 셔터가 중앙 기록 캡슐을 향해 닫혀 오는 장면으로 만들 것
- 5안은 하늘의 일식 원반과 아래 심장 제단을 하나의 축으로 연결할 것

### 게임 유저 에이전트

- 3안은 멋진 문은 보이지만 왜 지금 뚫어야 하는지가 약하다.
- 4안은 예쁜 UI처럼 보여 봉쇄 직전 탈취의 쾌감이 덜하다.
- 5안은 분위기는 좋지만 액션보다 상징이 먼저 보여 목표가 늦게 꽂힌다.

핵심 제안:
- 3안은 마지막 장갑 한 겹만 깨면 심장실이 열린다는 순간을 전면에 둘 것
- 4안은 지금 아니면 영영 코어를 못 가져간다는 상황을 보여줄 것
- 5안은 일식 창이 닫히면 세계가 영원히 잠긴다는 위기를 약속할 것
