실제 출시용 메인 아트 방향으로 압축한 1안과 3안

이번 라운드는 플레이 화면 느낌을 줄이고, 스토어 썸네일과 대표 홍보 아트까지 버틸 수 있는 `상징`, `실루엣`, `감정적 보상`을 강화했다. 1안은 요새 함락의 압도감, 3안은 일식을 가르고 세계의 빛을 되살리는 숭고함에 집중했다.

Main Art Criteria
1. 텍스트 없이도 무엇을 하는 게임인지 읽힐 것
2. 스토어 썸네일 크기에서도 핵심 실루엣이 살아남을 것
3. 플레이 화면이 아니라 포스터처럼 강하게 기억될 것
4. 성공 직전의 한순간을 잡아 감정적 기대를 만들 것
